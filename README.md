# Group-Work-Favorita-Forecasting


Time series analysis project using a modern Python `src` layout.
Includes tests, configuration files, notebooks, and modular source code.


### Init 

```bash
# init .venv
python -m venv .venv
```

```bash
# activ .venv windows
.venv\Scripts\activate
```

```bash
# activ .venv macOS
source .venv/bin/activate
```

```bash
# install dev
pip install -e .[dev]
```

```bash
# Hooks active
pre-commit install
```
